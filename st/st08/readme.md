# 计算器程序

### 添加的功能

1. 添加了除法运算
2. 支持输入小数点
3. 可以输入算式并支持删除单个字符（表达式解析的部分由AI编写，我进行修改和调整）
4. 无需按下`=`键即可实时显示结果
5. 按下`=`键后将结果显示在输入框中，并清空之前的算式
6. 美化界面（AI设计）
7. 支持键盘输入（AI编写，经过调整和修正）
